# Portfolio Website

Personal portfolio website for Satyam Warghat - Software Developer with 3+ years of experience.

## 🗂️ Project Structure

```
Portfolio/
├── index.html          # Main portfolio website
├── styles.css          # Styling for the portfolio
├── script.js           # JavaScript functionality (includes RSS feed integration)
├── profilephoto.png    # Profile photo
├── CLAUDE.md          # Claude Code instructions
│
├── assets/            # Images and media files
│   ├── about-image.png
│   ├── hero-image.png
│   ├── og-image.png
│   └── project-*.jpg  # Project screenshots
│
├── resumes/           # All resume versions
│   ├── resume.html
│   ├── resume-styles.css
│   └── various MD versions
│
├── backup/            # Old version backup (can be deleted when ready)
│
└── old_files/         # Unrelated files (safe to delete)
```

## 🚀 Quick Start

1. **Local Development**
```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```

2. **Deploy to Netlify**
- Drag and drop the folder to Netlify
- Custom domain setup available
